IP Address Tracker

IP Address Tracker uses several APIs like IP Geolocation API to fetch data about IP addresses and display them to the user, It's main features are providing information regarding the user's IP address on initial page load, and also allow the user to search an IP address and see similar information. Some of these information are the IP address location on a map, the ISP (Internet Service Provider), Timezone and so on.

Features

  • Intuitive user interface design.
  • Mobile first responsive design.
  • See own IP Address on the map on the initial page load
  • Search for any IP address.
  • Shows map location of searched IP address .
  • Shows the IP address issuing ISP .
  • Shows the timezone and city where the IP address is used.

Tools

  • React.js
  • TypeScript
  • SCSS
  • RESTful APIs
  • Map APIs

Locale - RESTful API

Locale is a developer tool for anyone who needs to know Nigeria, geographically at least. Locale’s API shows you all of Nigeria’s regions, states, and local government areas(LGAs). Locale is looking to be a very useful tool for the thousands of businesses building for Nigeria’s 200M+ population size.

Features

  • Users can register to get api key for accessing the API service.
  • Using the generated API Key, users can retrieve List of all regions, states and LGAs in Nigeria.
  • Using the generated API Key, users can search and retrieve metadata about regions, states and LGAs in Nigeria.

Tools

  • Node.js
  • TypeScript
  • Express.js
  • MongoDB
  • Json Web Token
